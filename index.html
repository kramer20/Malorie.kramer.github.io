Week Two of Twelve at
The Iron Yard
Cincinnati Campus

By Malorie Kramer, Student at The Iron Yard Cincinnati

This week we were introduced a new type of Framework, Bootstrap. Are main focus of Bootstrap now is learning how to use the grid system it provides. The grid system will allow me to more easily format columns within my website and ultimatley make things much easier.

We were asked to use Bootstrap to recreate a blog we had previously made using only CSS. The first thing that I changed was my column format. I used Bootstrap to create two columns, versus using multiple CSS properties to properly position my columns. Creating the callout section of my blog was much easier when I used the Bootstrap grid. I was able to properly place elements inline and used the offset to push the elements further left or further right. My search box was fairly simple to make and I was able to get the search symbol inside of the actual quiry. I read about the supported controls which have an input created for a search quiry. From input groups, I read how to use the button addons to create the search button with the symbol inside. I used the Bootstrap info option button to create my search button versus the general button used in HTML. The most I needed to use CSS for was to change fonts, colors, etc.
